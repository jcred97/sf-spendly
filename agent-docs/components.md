# Components

| Path                                           | Role                                                                                                                                                                                                                                                                                                                                                                               |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `lwc/spendlyApp`                               | Main workspace shell: auto-selected Expense Group context, collapsible desktop sidebar, group switcher, and data-driven internal views for Dashboard, Expenses, Recurring, and Settings; owns dashboard hero/insights/latest expenses, filters, summaries, charts, custom expense list, recurring template list/actions, delete, edit, duplicate, export, print, and modal trigger |
| `lwc/spendlyBarChart`                          | Reusable horizontal bar chart                                                                                                                                                                                                                                                                                                                                                      |
| `lwc/spendlyTrendChart`                        | Monthly trend visualization                                                                                                                                                                                                                                                                                                                                                        |
| `lwc/spendlySummaryCards`                      | Summary metric cards                                                                                                                                                                                                                                                                                                                                                               |
| `lwc/spendlyExpenseModal`                      | Add/Edit Expense modal: form, animations, focus trap                                                                                                                                                                                                                                                                                                                               |
| `lwc/spendlySettings`                          | Settings page for recurring automation controls and last-run status                                                                                                                                                                                                                                                                                                                |
| `classes/SpendlyController.cls`                | Apex backend: expense group/category/expense queries, dashboard trend query, recurring template overview query, and scoped DML actions                                                                                                                                                                                                                                             |
| `classes/SpendlySettingsService.cls`           | Singleton settings service and recurring run status tracker                                                                                                                                                                                                                                                                                                                        |
| `classes/SpendlyRecurringExpenseBatch.cls`     | Batch Apex recurring expense processor                                                                                                                                                                                                                                                                                                                                             |
| `classes/SpendlyRecurringExpenseScheduler.cls` | Daily scheduled Apex entry point                                                                                                                                                                                                                                                                                                                                                   |
| `classes/SpendlyRecurringExpenseService.cls`   | Manual recurring expense batch launcher                                                                                                                                                                                                                                                                                                                                            |
| `classes/SpendlyControllerTest.cls`            | Apex test class                                                                                                                                                                                                                                                                                                                                                                    |
